# 문제 : 가장 큰 수

1. 이해
    - 배열 안의 정수를 이어붙혔을 때가장 큰수가 나올 수 있도록 배치한다.
    - 이어 붙혔을때 가장 큰수가 되도록 하는 소트 방법은 ?
2. 계획
    - 두 수를 넣었을때 정렬하는 방법
        - 두 숫자를 앞과 뒤를 다른 순서로 이어 붙혀 두수를 비교 했을때 수가 더 크게 나오는 값의 앞의 수를 앞에 앞에 오도록 한다.
            - 두수를 입력 받는다.
            - 다른 순서로 이어 붙힌다.
            - 수를 비교한다.
            - 큰 값이 나오는 앞의 값이 앞으도록 -1을 리턴
